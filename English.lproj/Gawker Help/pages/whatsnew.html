What's new with Gawker

New features added to Gawker by version number:

New with Gawker Version 0.8.0

  • Users can now schedule start and stop times for their recordings
  • Camera sharing can be password-protected
  • Improved Timestamp appearance
  • User Interface improvements

New with Gawker Version 0.7.0

  • Compatible with more cameras! Tested mainly with DV FireWire cameras, your milage may vary. Compatible with macam for additional camera support.
  • Now handles multiple local cameras
  • Desktop timelapsing
  • Option to show countdown to next recorded frame
  • Slightly improved website

New with Gawker Version 0.6.2

  • Online Help
  • Automatic Checking for Updates
  • Thanks to Andy Matuschak for his Help Template and Sparkle!

New with Gawker Version 0.6.1

  • Fixed fetching of External IP address.

New with Gawker Version 0.6

  • Combined Camera rewrite, recording frame rate better when combining 2 network cameras.
  • User can now pick font for timestamp.
  • Users can get External and Internal IPs from the Share Options window(preliminary).
  • Timezone in timestamp now reported as GMT offset (i.e. GMT+7).
  • Can record movies at 320x240 or 640x480
  • Improved About Panel

New with Gawker Version 0.5.5

  • Universal Binary!
  • Now works with Built-in iSight

New with Gawker Version 0.5

  • Initial Release.
